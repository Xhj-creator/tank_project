4.18——4.20

刚开始忘记写开发日志了，具体哪一天干了什么不太记得了。这几天主要处理了玩家坦克的移动、射击，敌方坦克的随机移动、射击，子弹的
反弹、碰撞处理。这一段时间刚上手，对于unity预设体相关操作不太熟悉，导致我处理坦克旋转相关逻辑的时候非常费劲。美术组3D建模都
是以z轴竖直向上建立的，而且物体轴心没有设置在中心点上，且轴线方向与坦克朝向不平行，导致很多旋转函数都无法直接使用。原本想在
坦克下新建一个空白物体专门调整轴向，结果当时不会预设体解除操作导致预设体底下子物体不让拖拽，卡了我好久（）。随着制作的不断
进行，我逐渐了解了collider碰撞体，rigidbody刚体，layer层等多个组件的使用。刚开始移动全都是拿transform.translate写的，
结果发现碰撞和触发都完全无法监测，后面才切换到rigidbody的移动方式。接着又做了激光发射，武器触发切换，护盾等多个拾取道具的功
能，以及镜头跟随，血条显示等多个显示功能，搭建了整个场景的大致框架，做成了游戏操作界面的雏形。

4.21

用声波贴图的时候同样的图片类型却不会出现这个问题，最后实在没办法，只能用红蓝材质的墙代替激光墙。激光墙和声波的逻辑倒是挺简单，
主要是贴图卡了我好久。
